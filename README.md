Projeto de teste utilizando 
+ Spring Boot 
+ Spring Security
+ Spring Data JPA
+ Thymeleaf
+ Firebird 
+ bootstrap 
+ bootstrap-table 
+ server side pagination
+ angular 
+ bower

+ mvn spring-boot:run
+ mvn clean install
+ npm install -g bower
+ bower install
